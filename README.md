# 🎬 Friends - Aplicação Web Interativa

## Objetivo do projeto
Este é um projeto universitário que consiste no desenvolvimento de uma aplicação web interativa inspirada na série Friends. O objetivo é permitir que os usuários descubram com qual personagem mais se identificam por meio de um quiz.

Além do quiz, a plataforma também apresenta informações sobre a série, personagens e momentos marcantes, unindo entretenimento com tecnologia.

---

## Tecnologias Utilizadas
- HTML  
- CSS  
- JavaScript  
- Node.js  
- SQL  
- Chart.js
